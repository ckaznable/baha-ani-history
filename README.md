巴哈的觀看紀錄看起來有限制，太久以前看過的動畫紀錄看起來不會存起來，所以這個插件會把觀看紀錄存在瀏覽器帳號上

## Feature

* 看過的動畫集數會用橘色底
* 會自動補上最後一次看的集數
* 可以對集數標記顏色
